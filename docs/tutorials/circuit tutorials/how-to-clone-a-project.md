# How to clone an existing project

- Navigate to the **Projects** section.
![Circuit Tutorial 1](/img/Cloning-Tutorial/ct1.jpg)

- Go to the **Public Projects** tab.
![Circuit Tutorial 2](/img/Cloning-Tutorial/ct2.jpg)

:::info
For this tutorial, the **Two Stage Op-Amp** project is used as the reference example for cloning.
:::

- Search for **"Two Stage Op-Amp Demo."**
![Circuit Tutorial 3](/img/Cloning-Tutorial/ct3.jpg)

- Click on **Clone** for the selected project.
![Circuit Tutorial 4](/img/Cloning-Tutorial/ct4.jpg)

- Upon clicking **Clone**, you will be prompted to enter a name for the new project.
![Circuit Tutorial 5](/img/Cloning-Tutorial/ct5.jpg)

- After entering the name, the platform automatically creates and opens the cloned project.
![Circuit Tutorial 6](/img/Cloning-Tutorial/ct6.jpg)

- The cloned project will be available under the **My Projects** tab for further use and modification.
![Circuit Tutorial 7](/img/Cloning-Tutorial/ct7.jpg)

- The cloned project includes pre-configured circuits, allowing you to directly perform optimization using AI agents without needing to upload or set up the circuits manually.
