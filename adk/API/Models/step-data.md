# class StepData
A lightweight class used to represent the action taken by the agent at a particular time step.

## Definition
```py
class StepData:
    actions: NDArray
```

## Members
- ### `actions: NDArray`
    + **Description**: An NDArray representing the actions chosen by the agent for the current step.
